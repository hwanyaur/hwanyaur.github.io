Lumi Student Matcher
LumiNUS Tutors (Lumi for short) is a startup organization that provides free mentorship and tutoring to low-income high school students in Chicago. I joined Lumi when it was just an idea. I helped the 3 cofounders develop and complete our first-ever tutor program.

I helped create a program that matches tutors with students using data structures and algorithms on Python (pandas) to automate the pairing process and create the best possible matches.
